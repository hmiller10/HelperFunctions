# Invoke-ExpandZipArchive

## SYNOPSIS
.NET unzip function

## SYNTAX

### Set 1
```
Invoke-ExpandZipArchive [-ZipFileName] <String> [-DestinationDirectory] <String>
```

## DESCRIPTION
This function uses .NET to unzip a zip file to the specified location.
