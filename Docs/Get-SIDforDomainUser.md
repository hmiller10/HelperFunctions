# Get-SIDforDomainUser

## SYNOPSIS
Translate domain user to SID

## SYNTAX

### Set 1
```
Get-SIDforDomainUser [-Domain] <String> [-UserName] <String> [<CommonParameters>]
```

## DESCRIPTION
This function retrieves a domain user object SID using the Domain\\sAMAccountName
