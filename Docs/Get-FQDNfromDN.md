# Get-FQDNfromDN

## SYNOPSIS
Convert DN to FQDN

## SYNTAX

### Set 1
```
Get-FQDNfromDN [-DistinguishedName] <String> [<CommonParameters>]
```

## DESCRIPTION
This function converts an Active Directory distinguished name to a fully qualified domain name.
