# Get-DNfromFQDN

## SYNOPSIS
Get distinguished name from FQDN

## SYNTAX

### Set 1
```
Get-DNfromFQDN [-FQDN] <String> [<CommonParameters>]
```

## DESCRIPTION
This function translates a domain fully qualified domain name into its proper object distinguishedName

## EXAMPLES
